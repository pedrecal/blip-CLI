# Usage
<!-- usage -->
```sh-session
$ npm install -g blip-cli
$ blip-cli COMMAND
running command...
$ blip-cli (-v|--version|version)
blip-cli/0.0.0 linux-x64 node-v12.18.3
$ blip-cli --help [COMMAND]
USAGE
  $ blip-cli COMMAND
...
```
